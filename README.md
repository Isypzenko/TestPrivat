Тестовое задание

Проект заключается в возможности создавать и хранить баннеры. Баннеры содержат фото, описание. После создания можно редактировать и удалять их.

Использованный стек технологий.

- Angular
- Node JS
- Express
- MongoDB Atlas
- Moongoose for MongoDB

Инструкция по запуску

Перейдите в папку куда вы хотите склонировать проект.
Сделайте

- git clone https://github.com/Isypzenko/TestPrivat.git
- cd TestPrivat

Далее перейдите в папку client в терминале и запустите команду npm install.
Такую же процедуру проделайте и в папке server.
Далее в каждой папке в терминале введите команду npm run start. В консоле должны быть сообщения об успешном запуске и можете открыть проект.
